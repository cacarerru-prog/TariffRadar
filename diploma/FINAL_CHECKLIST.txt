✨ ПЕРЕД PUSH (финальные команды):

```bash
# 1. Проверить размер всего проекта
du -sh .

# 2. Посмотреть, что будет загружено
git status

# 3. Проверить, что диплом и примеры исключены
git check-ignore diploma/
git check-ignore практика/
git check-ignore пример-диплома/
# (не должно быть в git status)

# 4. Если всё OK — commit и push
git add .
git commit -m "Initial commit: TariffRadar MVP"
git push -u origin main
```
