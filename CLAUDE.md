## Суть проекта

**TariffRadar** — B2B двусторонний SaaS маркетплейс + аналитическая платформа для логистического рынка Беларуси. 

**Участники платформы:**
- **Грузоотправители** (физ. и юр. лица): размещают грузы для перевозки, ищут оптимальных логистов, анализируют рынок
- **Логисты/Перевозчики** (физ. и юр. лица): размещают услуги доставки, подбирают грузы, отслеживают спрос и конкурентные тарифы

**Модель монетизации (SaaS):**
- **Free:** базовый доступ (5 поисков/день, история 30 дней, базовая аналитика, без экспорта)
- **Pro ($29/месяц):** полный маркетплейс, история 90 дней, полная аналитика, экспорт CSV, без ограничений на поиск
- **Business ($99/месяц):** история 1 год, командный доступ (10 человек), API доступ, Slack/Telegram, приоритетная поддержка, автоматические отчеты

**Основные функции:**
- **Маркетплейс грузов:** размещение грузов и предложений логистических услуг, поиск контрагентов, прямые сделки
- **Аналитика рынка:** мониторинг тарифов по маршрутам, графики динамики цен, анализ трендов и сезонности
- **История сделок:** анонимный реестр с экспортом CSV для анализа тренда и бенчмарка
- **Поиск маршрутов:** фильтр по городам, типам груза, типам перевозки (FTL/LTL)
- **Добавление данных:** логисты добавляют информацию о выполненных перевозках для пополнения статистики (анонимно)
- **Система подписок:** Free, Pro ($29/месяц), Business ($99/месяц) с разным уровнем доступа к функциям

Расширяется как многопользовательская B2B2B платформа на весь рынок логистических услуг Беларуси с SaaS-монетизацией.

- **История данных:** ведение реестра сделок за 12+ месяцев, анонимная статистика, экспорт CSV

## Архитектура кода

**Слойная архитектура:**
- **Handlers** — HTTP-обработчики, валидация входных данных, форматирование ответов
- **Service** — бизнес-логика (создание грузов, поиск предложений, расчет аналитики)
- **Repository** — работа с БД (SQL-запросы, кэширование в Redis)
- **Models** — структуры данных для пользователей, грузов, предложений, сделок

**Основные бизнес-сущности:**
- `User` — пользователь (грузоотправитель или логист), профиль, реквизиты, текущая подписка
- `Plan` — тариф подписки (Free, Pro, Business) с набором функций и ограничениями
- `Subscription` — активная подписка пользователя (статус, дата начала/окончания, платежи через Stripe)
- `Cargo` — размещенный груз (маршрут, вес, объем, тип, сроки, статус)
- `Offer` — предложение логиста (цена, сроки, условия, компания)
- `Deal` — заключенная сделка между грузоотправителем и логистом (статус, доля в аналитике)
- `TariffData` — исторические данные о тарифах по маршрутам для аналитики и бенчмарка
